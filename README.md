# Cdaily

C작점 / Ctudy 의 C 스터디를 진행하면서 C언어의 문법을 공부하고

그와 동시에 프로그래밍 문제 풀이 연습을 하고자 합니다.

이 레포지터리에 올리는 문제들은 대부분 
* 작성자가 다른 과목을 공부하면서 드는 호기심을 해결하기 위함이거나
* 입시 수학 문제들을 프로그래밍으로 해결해보자

는 아이디어로 만들어졌습니다.
